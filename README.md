# BatteryThresholds 🔋

### en: Extension for Ubuntu (Gnome-shell 46) to control the minimum and maximum values at which the battery is charged or not charged. It works and is tested on laptop **Huawei Matebook 14 FLMH-X**

### ru: Расширение для Ubuntu (Gnome-shell 46) для контроля минимального и максимального значения при котором заряжается или не заряжается батарея). Работает и протестировано на ноутбуке **Huawei Matebook 14 FLMH-X**
